# Collections  

| Pattern | .Net Collection | Behavior |
|--------|-----------------|---------|
| Library Catalog | List<T> | Grow dynamically, orderes, index access |
| Return cart | Stack<T> | LIFO (puch, pop) |
| Customer Service Line | Queue<T> | FIFO (enque, deque) |
| Reading List | LinkedList<T> | Insert anywhere, reader |
